Take notes about interesting things you read about on the
internet without leaving your browser.

<!--more-->

## Goals

- Simple to use
- Intuitive interface
- Store notes in user's computer, not in servers

## Features

- [x] Select text on any website and provide a context
menu option to take a note (Completed 23 Jan 2026)
- [ ] Limit note size to 1 KB (~1000 characters)
- [ ] Store notes locally, on device
- [ ] Find notes by website, ordered by latest to oldest
- Rich text
    - [ ] Bold
    - [ ] Italic
    - [ ] Underlined
    - [ ] Strikethrough
    - [ ] Links
- [ ] Images (max 1 MB per image and one image per note)
